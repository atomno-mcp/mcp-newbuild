# atomno-mcp-newbuild

**Проверка новостройки и застройщика прямо в AI-ассистенте** — check Russian
new-build developers and construction projects for AI agents, straight from the
official ЕИСЖС / наш.дом.рф data (214-ФЗ). Works in Cursor, Claude Desktop,
Cline, and any MCP client.

Перед покупкой квартиры по ДДУ на миллионы рублей — за секунды проверьте
застройщика и объект, не листая вручную наш.дом.рф. Спросите ассистента
«надёжен ли застройщик по ИНН 7701234567?» или «что с ЖК по объекту OBJ-12345:
эскроу, разрешение, сроки?» — и получите структурированную справку из
первоисточника (ЕИСЖС): реквизиты застройщика, проектную декларацию, схему
эскроу, разрешение на строительство с историей переноса сроков и
детерминированную сводку риска.

> Данные носят **справочный** характер и **не являются инвестиционной
> рекомендацией**. Проверяйте первоисточник перед сделкой.

## Что умеет

- **Карточка застройщика** по ИНН/ОГРН — реестр застройщиков, объекты, число проблемных, фин.состояние по 214-ФЗ.
- **Проектная декларация** объекта — параметры дома, застройщик, банк, срок ввода, площади.
- **Проверка эскроу** — используются ли эскроу-счета (214-ФЗ) и уполномоченный банк.
- **Разрешение на строительство** — РНС, этап, срок ввода и **история переноса сроков**.
- **Каталог строящихся объектов** — фильтры по региону, застройщику, году ввода, стадии, эскроу.
- **Сводка риска застройщика** — детерминированная: доля проблемных объектов, средний сдвиг сроков, доля без эскроу.

## Быстрый старт

```bash
pipx install atomno-mcp-newbuild
# или: uvx atomno-mcp-newbuild
```

Cursor / Claude Desktop (`mcp.json` / `claude_desktop_config.json`):

```json
{
  "mcpServers": {
    "newbuild": {
      "command": "uvx",
      "args": ["atomno-mcp-newbuild"],
      "env": { "MCP_NEWBUILD_API_KEY": "<ваш-ключ-Pro>" }
    }
  }
}
```

## Переменные окружения

| Переменная | Описание | Обязательна |
|---|---|---|
| `MCP_NEWBUILD_API_KEY` | Ключ Pro (заголовок X-API-Key). [Получить](https://atomno-mcp.ru/pricing#newbuild-pro) | да |
| `MCP_NEWBUILD_API_BASE` | URL hosted-бэкенда (по умолчанию — прод) | нет |
| `MCP_NEWBUILD_TIMEOUT` | Таймаут HTTP, сек (default 30) | нет |

## Тулы

| Тул | Вход | Что возвращает |
|---|---|---|
| `check_developer` | ident (ИНН/ОГРН) | карточка застройщика: реквизиты, реестр, объекты, проблемные, фин.состояние |
| `get_project_declaration` | object_id | проектная декларация: параметры, застройщик, банк, срок ввода, площади |
| `check_escrow` | object_id | схема финансирования: эскроу/нет, уполномоченный банк |
| `check_construction_permit` | object_id | РНС, этап, срок ввода, история переноса сроков |
| `list_new_buildings` | region?, developer_inn?, commissioning_year?, stage?, escrow?, limit? | каталог строящихся объектов по фильтрам |
| `get_developer_risk_summary` | inn | доля проблемных, средний сдвиг сроков, доля без эскроу, введено в срок |

Каждый ответ содержит `source` (ЕИСЖС/наш.дом.рф), `as_of_date` и `disclaimer`.
